конкурентность - создаются потоки в одном процессе на одном ядре, с общим интерпретатором, и GIL, для выключения
одновременного доступа

Параллелизм (multiprocessing) - память разная для каждого процесса. - RAM, CPU мощность, etc., разные ядра
У одного процесса свой интерпретатор
между процессами нет GIL, внутри каждого процесса отдельно GIL есть
максимум процессов столько, сколько ядер, иначе включется конкуренция
